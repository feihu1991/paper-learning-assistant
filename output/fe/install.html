安装指南

本文档将帮助您在各种操作系统上安装和配置 OpenClaw。

环境要求

📋 系统要求
  • Node.js: 版本 22.0 或更高
  • 操作系统: macOS, Linux, Windows
  • 网络: 需要访问 openclaw.ai 下载安装脚本
检查 Node.js 版本 
node --version

如果未安装 Node.js 22+，请访问 nodejs.org 下载安装。

macOS / Linux 安装

💡 推荐： 使用官方安装脚本自动完成安装和配置。

一键安装

终端 
curl -fsSL https://openclaw.ai/install.sh | bash

安装过程说明

安装脚本会执行以下操作：

  1. 下载 OpenClaw CLI 工具
  2. 安装到系统路径
  3. 创建配置目录 ~/.openclaw/
  4. 配置环境变量
✅ 安装完成！ 接下来运行引导配置。

Windows 安装

⚠️ 要求： Windows 10/11，需要管理员权限的 PowerShell。

PowerShell 安装

PowerShell (管理员) 
iwr -useb https://openclaw.ai/install.ps1 | iex

手动安装（备选）

如果自动安装失败，可以手动配置：

  1. 确保已安装 Node.js 22+
  2. 全局安装 OpenClaw： 
    npm install -g @openclaw/cli
  3. 创建配置目录： 
    mkdir $env:USERPROFILE\.openclaw

引导配置

安装完成后，需要运行引导程序来配置守护进程和渠道连接。

运行引导程序

终端 
openclaw onboard --install-daemon

引导程序功能

  • 安装守护进程： 设置后台服务自动启动
  • 配置渠道： 连接 WhatsApp、Telegram、Discord 等
  • 设置代理： 配置 AI 模型提供商的 API Key
  • 创建工作区： 初始化第一个工作空间
💡 提示： 引导程序会交互式提示您完成配置，按照提示操作即可。

验证安装

安装完成后，请按以下步骤验证：

检查 Gateway 状态

检查状态 
openclaw gateway status

启动 Gateway

启动服务 
openclaw gateway start

打开 Dashboard

打开控制台 
openclaw dashboard
✅ 恭喜！ 如果服务正常启动，说明安装成功。浏览器会自动打开控制台界面。接下来请阅读 配置教程。

卸载 OpenClaw

卸载 
openclaw onboard --uninstall
