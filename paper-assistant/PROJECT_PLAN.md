# 论文学习助手项目计划

## 项目概述
创建一个网站，用户输入论文名称后，系统生成易于理解的学习内容和解释性图片。

## 技术栈
- **前端**: React + Tailwind CSS + Chart.js
- **后端**: Express.js + Node.js
- **数据库**: SQLite (开发) / MySQL (生产)
- **AI API**: OpenAI / DeepSeek API
- **图片生成**: DALL-E / Stable Diffusion API

## 功能模块
1. **论文搜索模块**
   - 论文标题输入
   - 实时搜索建议
   - 论文信息获取

2. **内容生成模块**
   - 摘要生成
   - 分步解释
   - 关键概念解释
   - 图表生成

3. **展示模块**
   - 响应式布局
   - 图片画廊
   - 交互式图表
   - 暗色/亮色主题

## 开发步骤
1. 搭建后端API框架
2. 实现AI API集成
3. 创建数据库结构
4. 开发前端界面
5. 集成前后端
6. 测试和优化
