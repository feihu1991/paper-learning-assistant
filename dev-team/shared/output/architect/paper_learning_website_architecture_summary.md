# 论文学习助手网站 - 架构设计摘要

## 核心架构决策
1. **前后端分离架构**: React前端 + Node.js后端
2. **微服务准备**: 清晰的服务边界，便于未来扩展
3. **多数据源集成**: arXiv、PubMed、IEEE等论文源

- **后端**: Node.js + Express + PostgreSQL
- **AI集成**: OpenAI GPT + 图表生成
- **部署**: Docker + Kubernetes
